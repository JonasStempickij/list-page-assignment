## How to build and run

Visual Studio Code C# extension required assets to build and debug are missing from... Add them?
**Yes**

Start API

1. `cd RecordShopAPI`
2. `dotnet run`

Start Front-end

1. `new terminal`
2. `cd RecordShopFE`
3. `npm install`
4. `npm run dev`
5. `goto http://localhost:3000`
